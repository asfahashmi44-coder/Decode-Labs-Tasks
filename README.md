# Decode Labs Tasks

This repository contains the codebase and deliverables for my programming tasks during my internship at DecodeLabs. It serves to track my progress, document project implementations, and maintain clean version control for all assigned exercises.

## 📂 Repository Structure

The project is structured as follows:

*   **`Task1.py`**: Implementation of the first assigned internship task (e.g., core logic, backend scripts, or initial data handling).
*   **`Task2.py`**: Implementation of the second phase or second standalone task (e.g., asynchronous operations, programmatic logic, or data tracking).
*   **`README.md`**: Project documentation and overview.
